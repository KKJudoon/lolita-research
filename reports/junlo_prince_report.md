### 1.2 商业模式与产业链

款的标准生命周期（来源：本项目内部 [industry-norms.md](https://github.com) 知识库整理 + [国牌 vs 日牌讨论 - 知乎多答主](https://www.zhihu.com/question/328712869) 交叉验证）：

```
画稿（画师，1次性买断 OR 分成+流团折现）
   ↓ 卖稿
品牌（电阻）打样
   ↓ 海报曝光（微博 / XHS）
意向金链上线（淘宝 ¥1-¥10 抵扣）
   ↓ 热度过线 → 启动正式预订
正式开订（开订-截单窗口，通常 1 个月）
   ↓ 截单
工厂打版 + 大货生产（3-12 个月）
   ↓ 出货 → 首发现货 / 二贩出物 / 闲鱼炒货
```

**钱流四档**：

| 阶段 | 性质 | 金额 |
|---|---|---|
| 意向金 | 抵扣（部分品牌可退） | ¥1 / ¥5 / ¥10 抵 ¥10 / ¥30 / ¥100 |
| 定金 | **不退** | ¥100-¥300 |
| 全款 / 通贩 | 不退 | OP ¥350-¥600 / 全套 ¥500-¥1000 |
| 二贩 | 闲鱼为主 | 浮动 |

**关键圈内规则**（每条 ≥ 2 证据交叉）：

- **画师 ≠ 设计师**——画师只画稿，品牌方做服装。圈内『买断 vs 分成』是两种品牌方与画师的分工模式。实证：[终焉圣骸 2.0 画师叁狐 XHS 04-10『平平无奇的军lo罢了 是我画的』](https://www.xiaohongshu.com/explore/69d885eb000000001a026e09)（画师本人发画稿声明，非品牌方）+ [龙曜的画师/品牌 IP 分工说明](items/yingyue_longyao.html)（设计帖）
- **流团**：款集结订单不够 → 取消生产，已付定金退还，画师按分成模式拿折现稿费
- **山店与厂原**：1:1 仿正版，价格 1/5。[圈内 2368 likes 高赞讨论『什么是厂原 lolita？厂原是山吗？』](https://www.xiaohongshu.com/explore/67aacfe00000000018009bc0) + [73 likes『关于LO为什么对山寨如此介意』莎娜可 2026-04-28](https://www.xiaohongshu.com/explore/69ef79cf000000002201f63e) + [344 likes『重复的已经不想截了，这lo裙到底怎么回事』2024-06](https://www.xiaohongshu.com/explore/665de127000000001401a040)（识图 30+ 家山店）— 三帖反映**山寨/厂原是圈内长期话题**
- **意向金抵扣 ≠ 真实销量**——淘宝 `¥5/¥1` 高销量榜单（如 [终焉圣骸 2.0 4w+ 付款 但全款待发](https://item.taobao.com/item.htm?id=1037713255465)）不能直接断言销量，要看微博/XHS 自然讨论度
