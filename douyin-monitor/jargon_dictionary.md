# 抖音 Lolita 画师圈黑话词典

最后更新：2026-05-11。原始样本：4 个 confirmed 画师（莓屿不语 / 无骨貓 / 仁儿这样来财 / 总裁化身裁缝版）+ 收藏夹 anchors（山福桃🍑 / 苹果鱼 / 鎏清 / 秘制弦菜鱼）+ 24 个 sig 黑话扫描通过的候选。

⚠ **核心价值**：这些词是**画师独有**，brand/wearer/tutorial **不会用**。出现 ≥ 2 个 → 高置信画师。这是把抖音 lolita 画师从噪音中分离的关键签名。

## 📐 综合打分算法

```python
def score(signature, nickname):
    s = 0
    # 高强词
    for kw in HIGH_SIGNALS: 
        if kw in signature: s += 10
    # 中强词
    for kw in MED_SIGNALS:
        if kw in signature: s += 5
    # 弱词
    for kw in WEAK_SIGNALS:
        if kw in signature: s += 2
    # 昵称信号
    for suffix in NICK_SUFFIXES:
        if suffix in nickname: s += 8
    # 反信号扣分
    for kw in TUTORIAL_SIGNALS:
        if kw in signature: s -= 12
    return s
```

**阈值**：
- ≥ 30：极强候选，几乎必是画师
- 15-29：强候选，多数是画师
- 10-14：中候选，需要看作品验证
- 5-9：弱候选，先存
- <5：丢弃
